# Analisis-Ventas-Supermercado

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un dashboard interactivo creado para analizar el rendimiento comercial de una cadena de supermercados. El objetivo principal es transformar un conjunto de datos transaccionales en insights visuales claros que faciliten la toma de decisiones, mostrando el comportamiento de los ingresos, el perfil de los compradores y el rendimiento de los productos en distintas sucursales.

## Herramientas Utilizadas
*   **Herramienta de BI:** Power BI Desktop.
*   **Lenguaje de Fórmulas:** DAX (Data Analysis Expressions) para la creación de medidas personalizadas y KPIs (Total Ventas, Total Ingresos).
*   **Fuente de Datos:** Archivo estructurado `.csv` con registros detallados de transacciones.

## Insights y Visualizaciones Clave
El dashboard está estructurado como un Resumen Ejecutivo que permite filtrar dinámicamente por ciudad, tipo de cliente, género y categoría. Incluye:
*   **Métricas de Alto Nivel:** Tarjetas de KPIs con el volumen total de ventas y los ingresos generados.
*   **Análisis Geográfico:** Gráfico de columnas que compara los ingresos totales entre las sucursales (Nueva York, Chicago y Los Ángeles).
*   **Segmentación de Clientes:** Gráficos de anillos que desglosan la participación de ingresos por clientes "Miembros" vs. "Normales" y por género.
*   **Rendimiento de Inventario:** Gráfico de barras horizontales que identifica las categorías de productos más rentables.
